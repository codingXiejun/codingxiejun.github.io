前言

该项目已上传至Github，下载地址
使用SpringBoot可以轻松地搭建一个新项目免去了许多繁琐的配置过程

而以前我们搭建一个简单的ssm项目，结构是这样的：

我们不但要配置一堆的配置文件，还需要搭建Tomcat服务器，还需要自己上网下载需要的jar包，下载包就算了，还有各种包冲突……

有了springboot，我们只需要非常简单的配置就可以成功搭建一个JavaWeb项目。

springboot是什么

springboot是一个微服务框架，它被用来简化创建Spring应用的初始搭建和开发过程，它继承了原有Spring框架的优秀基因，使编码变得更加简单。

使用springboot的好处

不必每次创建项目都配置一遍，降低开发成本并且更加简洁可靠~

开始第一个项目

下载SpringBoot插件

本人使用的IDE是eclipse，所以就以eclispe为例子。

  • 打开eclipse下载商城
    Help -> Eclipse User Storage -> Open Marketplaces Favorites…
    搜索Spring Tool然后进行下载安装

打开Springboot官网


(依赖可以后面再添加)点击Generator然后下载到本地就行了

导入项目

  • File -> Import -> Maven -> Exist Maven project -> Next -> Browser
  • 等待eclipse下载完所需要的Jar包
    下载完后位移的一个java文件就是项目的入口，所有文件都必须在它同名包下才能成功运行，所以一般我们需要新建一个上级包来放置它
  • 新创建一个上级包，将主函数放到其中

新建HelloController类放到controller包下

HelloController代码：

@RestController：直接将字符串转换成JSON对象返回

运行主函数

Tips：

  • SpringBoot集成了Tomcat服务器，并且访问不需要加项目名
  • 可以在主函数添加Print语句来更直观地检查是否启动成功
  • 每次运行主函数，都要打开任务管理器关掉javaw.exe,不然会报错

访问页面

浏览器输入localhost:8080/hello

访问成功！

结语

搭建一个SpringBoot项目从头到尾不超过十分钟，效率提高了好几倍，不过对于初学者，还是得先打好Spring框架的基础再来使用它，不要本末倒置
